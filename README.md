# Habit-Tracker
This app is based on node js , express , mongoose
* Host Link :- https://habit-tracker98.herokuapp.com/

# Features
* Add Habit
* You can add habit to favourite and also remove from favourite.
* Delete 
* Yo can mark as complete , incomplete , pending;
